CINÉ DÉLICES

L'idée est née d'une question simple à O'Clock : "et si chaque film avait son menu ?" J'ai conçu le design system avant d'écrire une ligne de code : typographies Cinzel + Playfair, palette or/noir/rouge, badges animés. Premier vrai projet full-stack où j'ai géré seul la partie front intégrale.
